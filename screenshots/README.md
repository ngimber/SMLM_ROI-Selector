# GUI Screenshots

## Main Interface

**File: gui_screenshot.png**

![ROI Selection Tool GUI](https://raw.githubusercontent.com/ngimber/SMLM_ROI-Selector/main/screenshots/gui.png)

This screenshot shows the main interface of the SMLM ROI Selector tool displaying;
- The grayscale localization image in the center
- Control panels on the left side (Channel, Pixel size, Draw Mode, etc.)
- Action buttons at the bottom (Load File, Save, Clear ROIs, Help)
- An example ROI drawn with red outline

## Features Shown

- **Loaded Data**: Localization file with visible molecular cluster structure
- **ROI Selection**: Red outlined region showing the current or completed selection
- **Controls**: All adjustment sliders and radio buttons
- **Contrast Visualization**: Image with clear visibility of localizations
